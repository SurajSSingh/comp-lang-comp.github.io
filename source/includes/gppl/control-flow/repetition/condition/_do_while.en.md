#### Do While Loop

```csharp
bool condition = true;
do {
    // Run this code at least once
}
while(condition);
```

```python
ran_once = false
condition = True
while condition or not ran_once:
    # Run this code at least once
    # Update ran once at the end of the loop
    ran_once = True
```

```rust
let condition = true;
loop {
    // Run this code at least once
    // Break
    if !condition {break}
}
```
