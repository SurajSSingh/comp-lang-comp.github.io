## Line Comments

Line comments are comments that run from the start of some symbol to the end of the line.

> Line Comments between language

```csharp
// This is a single line comment
```

```python
# This is a single line comment
```

```rust
// This is a single line comment
```


| Start Symbol | Languages |
| -------------- | ----------- |
| //           | C#, Rust  |
| #            | Python    |
